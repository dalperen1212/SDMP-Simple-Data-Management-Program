# SDMP-Simple-Data-Management-Program
SDMP: Simple Data Management Program

SDMP, bir işe alım yöneticisinin aday verilerini bir dosya (veritabanı) üzerinde saklamasına ve bu verileri yönetmesine olanak tanıyan basit bir yönetim simülasyonudur.
--Projenin Amacı

Bu proje, temel veri yönetimi işlemlerini (CRUD) simüle etmek amacıyla geliştirilmiştir. Kullanıcıların aday bilgilerini kalıcı bir metin dosyasında depolamasını ve terminal üzerinden bu verileri manipüle etmesini sağlar.

    Not: Temel düzeydeki ihtiyaçlar için kullanıma uygundur ancak profesyonel ölçekli sistemlerin yerini tutma amacı gütmemektedir.

--Kullanılan Teknolojiler

    Dil: Python 3.x

    Veri Saklama: Flat-file (TXT tabanlı veritabanı)

--Kurulum

Projeyi yerel makinenize indirmek için terminale şu komutu yazabilirsiniz:
Bash

git clone https://github.com/dalperen1212/SDMP-Simple-Data-Management-Program.git

--Kullanım

    Terminal üzerinden projenin indirildiği dizine gidin.

    Program klasörüne giriş yapın:
    Bash

    cd Program

    Uygulamayı başlatın:
    Bash

    python3 main.py

    Terminal ekranındaki yönergeleri takip ederek veri ekleyebilir veya mevcut verileri yönetebilirsiniz. Tüm değişiklikler anlık olarak xxx.txt dosyasına yansıtılacaktır.

--Proje İstatistikleri

Bu proje, yazılım temellerini pekiştirmek amacıyla hızlı bir geliştirme süreciyle (Rapid Prototyping) hazırlanmıştır.

    Geliştirme Süresi: ~36 Dakika (PyCharm - WakaTime verilerine göre)

    Odak Noktası: Dosya yönetimi ve Terminal UI mantığı.
